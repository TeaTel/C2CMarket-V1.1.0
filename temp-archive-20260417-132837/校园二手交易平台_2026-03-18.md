# 校园二手交易平台开发日志

日期：2026-03-18

## 本次改动内容

### 1. 环境初始化
- 创建了 `env-setup` 文件夹用于环境初始化
- 初始化了 Git 仓库
- 创建了 Spring Boot 后端工程，集成了 `web`、`mysql`、`mybatis`、`spring-security` 等基础依赖
- 创建了 Vue3 前端工程，集成了 `vue`、`vue-router`、`axios` 等基础依赖
- 配置了 `.gitignore` 文件，忽略不需要提交的文件

### 2. 后端通用组件
- 在 `common` 包下创建了 `Result<T>` 泛型类，包含 `code`、`message`、`data` 字段
- 提供了 `success()` 和 `error()` 静态方法，用于统一响应格式
- 创建了 `GlobalExceptionHandler` 类，使用 `@RestControllerAdvice` 拦截所有异常并返回 `Result` 格式

### 3. 接口文档配置
- 在 `pom.xml` 中添加了 knife4j (Swagger 增强版) 依赖
- 在 `config` 包下创建了 `SwaggerConfig` 配置类，开启接口文档
- 设置了分组为“二手交易平台”，方便管理接口文档

### 4. 商品列表接口实现
- 创建了完整的代码链路：Controller → Service → Mapper → DTO
- **Controller**：实现了 `GET /api/products` 接口，支持分类筛选、关键词搜索、分页查询
- **Service**：实现了 `ProductService` 接口，调用 Mapper 查询数据并转换为 VO
- **DTO**：创建了 `ProductQueryDTO` 用于接收查询参数，`ProductVO` 用于返回数据
- **Mapper**：创建了 `ProductMapper` 接口和 XML 映射文件，实现了商品列表查询的 SQL 语句
- **Entity**：创建了 `Product` 实体类，对应数据库中的 `products` 表
